## 📈 Understanding the features

The software provides several tools to help you manage transaction security.

### Live transaction monitoring
The app connects to your data stream to watch transactions in real time. It marks any transfer that looks unusual. You can see these alerts on the main dashboard immediately.

### Network link analysis
Graphs show how accounts interact. The system uses a method called GraphSAGE to map these links. This allows the software to see chains of accounts that might belong to one bad actor.

### Fraud risk scoring
Every transaction receives a score. Low scores mean the transaction looks normal. High scores mean the transaction carries a high risk. You can prioritize your review based on these scores.

## 📖 Glossary of terms

- Data stream: The incoming flow of transaction information into the app.
- Graph: A way of drawing connections between different accounts and money transfers.
- Neural network: A math model that learns to spot fraud by looking at past examples.
- Risk score: A number that tells you how likely a transaction is to be fraud.
